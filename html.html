Wat is HTML?

Intro in 3 front end technologieën

  • Content laag, informatie die je ziet op een pagina > HTML
  • Presentatie laag, hoe ziet de informatie eruit? > CSS
  • Gedrag laag, zorgt voor interactie met de pagina

HyerText Markup Language

HTML is een opmaak taal en zorgt voor de basisstructuur van een website. Bepaald bijvoorbeeld wat headers, paragrafen en tabellen zijn. Elke website is gemaakt met HTML.
